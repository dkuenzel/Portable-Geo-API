Format for peer 2 peer requests: "/p2p/integer:outputFormat/float:origin_lon/float:origin_lat/float:destination_lon/float:destination_lat"
Where:
outputFormat =
 0) Beautified (HTML)
 1) Raw output (JSON)
 2) Route only (returned as array of wkt elements with EPSG 4326)
 3) Distance only

Format for isochrone requests: "/ich/integer:outputFormat/float:maxRange/float:origin_lon/float:origin_lat"
Where:
maxRange = distance in hours
outputFormat =
 0) Beautified (HTML)
 1) Raw output (JSON)
 2) Nodes only (JSON)
 3) Geometry only (returned as WKT element with EPSG 4326)
